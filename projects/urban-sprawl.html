Urban Sprawl Analysis - Nairobi, Kenya

Project Overview

Monitored urban expansion over 20 years using Landsat imagery, providing insights for sustainable city planning and infrastructure development.

Methodology

  • Landsat 5, 7, and 8 satellite imagery (2000-2020)
  • Supervised classification using Random Forest algorithm
  • Change detection analysis
  • Accuracy assessment: 92% overall accuracy

Key Findings

  • Urban area expanded by 156% between 2000-2020
  • Agricultural land decreased by 45%
  • Informal settlements grew by 200%
